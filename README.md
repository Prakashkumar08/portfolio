# Prakash Kumar G - IoT Developer Portfolio

A modern, animated portfolio website with IoT-themed design and multiple color themes.

## Features

### 🎨 Multiple Themes
- **IoT Dark** (Default) - Cyan and purple cyberpunk theme
- **Cyber Light** - Light mode with ocean blue accents
- **Purple Haze** - Deep purple and violet theme
- **Ocean Blue** - Aqua and deep sea colors
- **Forest Green** - Nature-inspired green theme
- **Sunset Orange** - Warm orange and fire tones

### ✨ Animations
- Animated circuit board background
- Typing effect for job titles
- Smooth scroll animations
- Counter animations for statistics
- Progress bars for skills
- Floating IoT icons
- Parallax effects

### 📱 Responsive Design
- Mobile-friendly navigation
- Adapts to all screen sizes
- Touch-optimized interface

### 🔧 Sections
- Hero section with profile image
- About with animated statistics
- Work experience timeline
- Education cards
- Projects showcase
- Technical skills with progress bars
- Contact information

## How to Use

1. Open `index.html` in any modern web browser
2. Click the palette icon (🎨) in the navigation bar to switch themes
3. Your theme preference is automatically saved

## Theme Persistence

The selected theme is saved in your browser's localStorage and will be remembered on your next visit.

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript
- Canvas API for animated background
- Font Awesome for icons

## Customization

To add more themes, edit the CSS variables in `style.css` and add corresponding theme options in `index.html` and `script.js`.
